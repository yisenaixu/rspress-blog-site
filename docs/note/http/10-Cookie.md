# Cookie

HTTP是无状态的，而Cookie给HTTP增加了记忆能力

## 什么是Cookie

> 相当于是服务器给每个客户端都贴上一张小纸条，上面写了一些只有服务器才能理解的数据，需要的时候客户端把这些信息发给服务器，服务器看到 Cookie，就能够认出对方是谁了。

## 工作过程

### Set-Cookie 响应字段

服务器创建的对客户端的身份标识数据

### Cookie 请求字段

浏览器发送请求时携带的数据

1. 浏览器第一次发送请求给服务器
2. 服务器创建一个身份表示数据放入Set-Cookie字段发给浏览器
3. 浏览器收到后把值保存，之后发送请求都会自动把值放入Cookie发送给服务i去
4. 之后服务器可以识别用户身份
